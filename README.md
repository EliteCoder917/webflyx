# FlowBot 

FlowBot is a simple productivity Discord bot designed to help users focus, stay consistent, and build better habits.

## Features (V1)
- /focus [minutes] – start a focus timer  
- /break [minutes] – start a break timer  
- /streak – track daily focus streaks  
- /quote – get a short motivational message  

## Tech Stack
- Python
- discord.py
